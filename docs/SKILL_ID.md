# Tailwind CSS – Platform Ujian

## Filosofi UI

- Dashboard bersih (admin, guru, fkkg)
- Mode ujian minim distraksi
- Mudah dibaca
- Responsif & realtime

---

## Modul UI

### 1. Dashboard
- Sidebar dinamis (role-based)
- Topbar (status, notif)
- Card statistik

---

### 2. UI Ujian (KRITIKAL)
- Fullscreen
- Minim scroll
- Navigasi soal sticky

---

### 3. Panel Soal
- Typography jelas
- Opsi jawaban besar

---

### 4. Navigasi Jawaban
- Nomor soal grid
- Status warna:
  - terjawab
  - ditandai
  - kosong

---

### 5. Indikator Realtime
- Status koneksi
- Timer sinkron

---

### 6. Dark Mode
- Wajib support

---

## Aturan UX

- Tidak ada animasi berlebihan
- Tidak ada layout shift
- Tidak ada popup tiba-tiba

---

## Performa

- Hindari DOM berat
- Gunakan gap, bukan margin
